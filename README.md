<h1>💱 Conversor de Moedas Internacional</h1>

<p>
Este foi o meu primeiro projeto utilizando 
<strong>HTML</strong>, <strong>CSS</strong> e <strong>JavaScript</strong> juntos, 
desenvolvido com foco em praticar lógica de programação, manipulação do DOM 
e criação de interfaces modernas e responsivas.
</p>

<p>
O projeto consiste em um conversor de moedas internacionais com interface moderna, 
permitindo converter valores entre diferentes moedas de forma dinâmica e automática.
</p>

<h2>🚀 Tecnologias Utilizadas</h2>

<ul>
    <li>HTML5</li>
    <li>CSS3</li>
    <li>JavaScript</li>
</ul>

<h2>📚 O que estudei neste projeto</h2>

<ul>
    <li>Estruturação semântica com HTML</li>
    <li>Responsividade para mobile e desktop</li>
    <li>Flexbox e posicionamento com CSS</li>
    <li>Gradientes, sombras e design moderno</li>
    <li>Manipulação do DOM</li>
    <li>Eventos em JavaScript</li>
    <li>Captura de valores com input e select</li>
    <li>Conversão dinâmica entre moedas</li>
    <li>Objetos em JavaScript</li>
    <li>Fórmula universal de conversão</li>
    <li>Intl.NumberFormat()</li>
    <li>Formatação internacional de moedas</li>
    <li>Controle de fluxo com if</li>
    <li>Validação de inputs</li>
    <li>Funções reutilizáveis</li>
    <li>Lógica matemática aplicada</li>
</ul>

<h2>🎯 Objetivo</h2>

<p>
O objetivo principal foi aprender como integrar as três tecnologias principais 
do Front-End em um projeto real, aplicando design moderno e lógica funcional ao mesmo tempo.
</p>
